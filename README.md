📌 Project Description – Smart Task & Habit Tracker

The Smart Task & Habit Tracker is a simple web application developed using HTML, CSS, Bootstrap, and JavaScript. It helps users organize and monitor their daily activities effectively. The main features of this project are:

Add New Tasks – Users can add tasks with a title, description, category, date, and status (Pending, Completed, or Overdue).

Display in Cards – Each task is shown in a Bootstrap card with proper styling for better readability.

Mark as Completed – Tasks can be marked as “Completed” with a single click.

Delete Tasks – Unwanted tasks can be removed from the list.

Search Functionality – Users can search tasks by title, description, category, or date.

Filter Option – Tasks can be filtered based on their status (All, Completed, Pending, Overdue).

Dashboard – A live dashboard shows:

Total tasks

Completed tasks

Pending tasks

Overdue tasks

Progress percentage (calculated from completed tasks).

Responsive UI – The design uses Bootstrap for a clean and user-friendly interface.
